# Tostato

Sito one-page per Tostato — Toasted Bread & Coffee Bar.

## Struttura

- `index.html` — markup della pagina (hero, chi siamo, menu, galleria, contatti)
- `css/style.css` — stile e animazioni
- `js/main.js` — barra di progresso, nav allo scroll, menu mobile, reveal allo scroll
- `assets/img/` — immagini del locale

## Palette

Ispirata alle foto reali del locale: navy/antracite come base scura, senape/oro come accento, mattone a vista e legno chiaro.

## Sviluppo locale

Sito statico, nessuna build richiesta:

```
python3 -m http.server 8000
```

poi apri `http://localhost:8000`.
